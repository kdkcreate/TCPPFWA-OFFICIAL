Sensor Documentation

DHT11 Sensor

The DHT11 sensor is a low-cost digital device that measures temperature and relative humidity.

Working Principle

It uses a capacitive humidity sensor and thermistor to produce a calibrated digital output signal.

Ultrasonic Sensor (HC-SR04)

The ultrasonic sensor measures distance by emitting sound pulses and measuring echo return time.

Working Principle

Uses transmitter and receiver to calculate distance based on sound speed.

Sound Sensor

The sound sensor detects environmental sound intensity.

Working Principle

Converts acoustic signals into electrical signals for monitoring.

Gas/Smoke Sensors (MQ Series)

Detects harmful gases and smoke, providing early hazard warnings.

Working Principle

Heated semiconductor changes resistance when exposed to gases, outputting analog voltage.

Board Integration

The ESP8266/ESP32 connects sensors and transmits data over Wi-Fi. It communicates with platforms like Adafruit IO or Blynk using MQTT or HTTP.

  • Ensure adequate power supply.
  • Avoid GPIO conflicts.
  • Install required libraries like Adafruit_Sensor, DHT, PubSubClient, and BlynkSimpleEsp8266.

Sensors not reading: Check wiring and pins; add sensor delays.
